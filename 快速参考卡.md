## 📋 选题评分卡

```
□ 专业深度：___/10 （你对这个话题的理解程度）
□ 受众广度：___/10 （有多少人关心这个话题）
□ 时效热度：___/10 （现在是不是讨论的好时机）

✅ 及格线：≥21分，且单项≥6分
```

**快速判断：**
- 27-30分：优秀选题，立即开写
- 21-26分：合格选题，可以写
- 15-20分：需要优化角度
- <15分：放弃或重新选题

## 📝 创作流程清单

### 阶段1：选题分析 ⏱️ 30分钟
```
□ 确定主题
□ 评分（专业度+受众度+时效度）
□ 竞品分析（找3-5篇相关文章）
□ 确定差异化角度
```

### 阶段2：大纲生成 ⏱️ 1小时
```
□ 设计3个备选标题
□ 写开头（钩子设计）
□ 列出3-5个主要部分
□ 每部分列出核心观点+案例
□ 设计结尾（总结+行动召唤）
□ HKR检查
```

### 阶段3：内容创作 ⏱️ 3-5小时
```
□ 按大纲创作
□ 每300字有1个价值点
□ 每500字有1个案例/故事
□ 控制段落长度（手机屏2-3屏）
□ 关键观点加粗
```

### 阶段4：质量检查 ⏱️ 30分钟
```
□ 检查错别字和语病
□ 检查逻辑是否严密
□ 检查数据是否准确
□ HKR评分
□ 爆款要素检查
```

### 阶段5：优化迭代 ⏱️ 1小时
```
□ 优化标题（A/B测试）
□ 优化开头（前3句话）
□ 调整结构
□ 增加细节
□ 强化情感
```

## 🎯 爆款要素检查表

### 必备要素（至少满足5项）
```
□ 有争议性/反常识观点
□ 有真实的个人经历/案例
□ 有具体的数据支撑
□ 有实用的方法/工具
□ 有情感共鸣点
□ 有传播价值（读者愿意分享）
□ 有讨论空间（能引发评论）
```

### 加分要素（满足越多越好）
```
□ 蹭到热点/趋势
□ 填补了信息空白
□ 颠覆了常见认知
□ 提供了新视角
□ 解决了痛点问题
□ 引发了情感共鸣
□ 具有收藏价值
```

### 减分要素（必须避免）
```
□ 标题党（内容配不上标题）
□ 逻辑混乱
□ 观点陈旧
□ 内容空洞
□ 自我感动
□ 过度煽情
□ 明显错误
```

## 💡 标题公式库

### 公式1：数字型
```
《我用这N个方法，实现了XXX》
《关于XXX的N个真相》
《N年XXX经验，总结出这N条》
```

### 公式2：反差型
```
《你以为的XXX，其实是YYY》
《别再XXX了，真相是YYY》
《XXX不是YYY，而是ZZZ》
```

### 公式3：利益型
```
《掌握这个方法，你也能XXX》
《做到这N点，XXX不再难》
《XXX的正确打开方式》
```

### 公式4：悬念型
```
《为什么XXX？答案让我震惊》
《关于XXX，我发现了一个秘密》
《XXX背后的真相》
```

### 公式5：共鸣型
```
《如果你也XXX，一定要看这篇》
《XXX的人，都有这N个特征》
《做XXX这么久，终于想明白了》
```

## ⚡ 快速优化技巧

### 标题不够吸引？
```
✅ 加数字："10个"、"3年"、"90%"
✅ 加悬念："真相"、"秘密"、"发现"
✅ 加利益："涨薪"、"月入"、"提升"
✅ 加冲突："别再"、"不是"、"其实"
```

### 开头不够抓人？
```
✅ 第一句话制造冲突/悬念
✅ 前3句话必须有价值
✅ 不要铺垫太多背景
✅ 直接切入核心
```

### 内容太干巴巴？
```
✅ 增加个人经历
✅ 增加具体案例
✅ 增加情感表达
✅ 使用比喻和类比
```

### 结尾太平淡？
```
✅ 总结核心观点（1-3条）
✅ 升华主题
✅ 给出行动建议
✅ 引导评论互动
```

## 🚨 常见错误清单

### 选题阶段
```
❌ 选题太大太空
❌ 选题太小众
❌ 选题没有时效性
❌ 选题自己不懂
```

```

❌ 全是理论没有案例
❌ 自我感动式抒情
❌ 逻辑跳跃

```
❌ 不检查错别字
❌ 不优化标题
❌ 不做HKR检查
❌ 不看数据反馈
