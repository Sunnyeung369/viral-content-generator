## 核心方法论

### 一、选题三要素模型
```
好选题 = 专业领域 ∩ 读者兴趣 ∩ 时间节点
```

**评分标准：**
- 专业深度（0-10分）：你对这个话题的理解程度
- 受众广度（0-10分）：有多少人关心这个话题
- 时效热度（0-10分）：现在是不是讨论的好时机

**及格线：** 总分≥21分，且单项≥6分

### 二、HKR内容质量模型
```
优质内容 = Hook(钩子) + Knowledge(知识) + Resonance(共鸣)
```

**评分标准：**
- H（吸引力）：标题、开头、悬念设置（0-10分）
- K（价值度）：信息量、洞见、实用性（0-10分）
- R（共鸣度）：情感连接、代入感、认同感（0-10分）

**及格线：** 总分≥24分，且单项≥7分

### 三、内容层次金字塔
```
顶层：核心观点（1个）
  ↓
中层：支撑论据（3-5个）
  ↓
底层：案例故事（若干）
```

### 四、读者画像公式
```
理想读者 = 聪明（能理解） + 有钱（有选择） + 很忙（重效率）
```

**输出：**

```markdown
## 选题评分
- 专业深度：X/10
- 受众广度：X/10
- 时效热度：X/10
- 总分：XX/30

## 竞品分析
- 已有内容：[列出3-5篇相关文章]
- 差异化角度：[你的独特视角]

## 关键词云
[核心关键词、长尾关键词]

## 建议
- 推荐角度：XXX
- 风险提示：XXX
- 优化建议：XXX
```

**输入：**
- 选题分析结果

- 目标字数

**输出：**
```markdown

## 开头（钩子设计）
[200字左右的开头，包含悬念/故事/观点]

## 主体结构
### 第一部分：[小标题]
- 核心观点：XXX
- 支撑论据：XXX
- 案例故事：XXX
- 预计字数：XXX

### 第二部分：[小标题]
...

## 结尾（行动召唤）
[总结+升华+引导互动]

## HKR检查
- H（钩子）：[设计说明]
- K（知识）：[价值点列表]
- R（共鸣）：[情感连接点]
```

### 阶段3：内容创作

**创作原则：**

1. **开头黄金30秒**
   - 前3句话必须抓住注意力
   - 使用：悬念/冲突/反常识/故事/数据

2. **主体信息密度**
   - 每300字必须有1个价值点
   - 每500字必须有1个案例/故事
   - 避免废话和自我感动

3. **节奏控制**
   - 长短句结合
   - 段落长度适中（手机屏2-3屏）
   - 适时使用小标题分段

4. **情感波动**
   - 制造起伏，不要平铺直叙
   - 在关键观点前制造悬念
   - 在案例中注入情感

5. **结尾强化**
   - 总结核心观点（1-3条）
   - 升华主题
   - 引导行动/互动

### 阶段4：质量检查

**检查清单：**

#### 内容层面
- [ ] 核心观点清晰明确
- [ ] 每个论据都有支撑
- [ ] 案例真实可信
- [ ] 数据准确有来源
- [ ] 逻辑严密无漏洞

#### 表达层面
- [ ] 无错别字
- [ ] 无语病
- [ ] 无歧义
- [ ] 标点符号正确
- [ ] 排版美观

#### 价值层面
- [ ] 有新信息/新角度
- [ ] 有实用价值
- [ ] 有情感共鸣
- [ ] 有行动指引
- [ ] 有传播价值

#### HKR评分
- [ ] H（吸引力）≥7分
- [ ] K（价值度）≥7分
- [ ] R（共鸣度）≥7分
- [ ] 总分≥24分

### 阶段5：优化迭代

**优化方向：**

1. **标题优化**
   - A/B测试3个版本
   - 加入数字/悬念/利益点
   - 控制在20-30字

2. **开头优化**
   - 前3句话反复打磨
   - 确保钩子够强
   - 测试不同开头方式

3. **结构优化**
   - 检查逻辑流畅性
   - 调整段落顺序
   - 增删小标题

4. **细节优化**
   - 替换平淡的词汇
   - 增加具体的细节
   - 强化情感表达

5. **互动优化**
   - 增加提问
   - 设置讨论点
   - 引导评论

## 高级技巧

### 技巧1：标题公式库

**数字型：**
- 《我用这N个方法，实现了XXX》
- 《关于XXX的N个真相》
- 《N年XXX经验，总结出这N条》

**反差型：**
- 《你以为的XXX，其实是YYY》
- 《别再XXX了，真相是YYY》
- 《XXX不是YYY，而是ZZZ》

**利益型：**
- 《掌握这个方法，你也能XXX》
- 《做到这N点，XXX不再难》
- 《XXX的正确打开方式》

**悬念型：**
- 《为什么XXX？答案让我震惊》
- 《关于XXX，我发现了一个秘密》
- 《XXX背后的真相》

**共鸣型：**
- 《如果你也XXX，一定要看这篇》
- 《XXX的人，都有这N个特征》
- 《做XXX这么久，终于想明白了》

### 技巧2：开头钩子库

**悬念式：**
> "我花了三年时间，才发现这个被99%的人忽略的真相……"

**冲突式：**
> "所有人都说要XXX，但我要告诉你，这是个谎言。"

**故事式：**
> "2023年的那个夜晚，我做了一个改变我人生的决定……"

**数据式：**
> "根据最新调查，73%的人都在XXX，但只有3%的人知道YYY。"

**提问式：**
> "你有没有想过，为什么XXX？今天我就来揭开这个谜底。"

### 技巧3：过渡句式库

**承上启下：**
- "说完了XXX，我们再来看YYY"

- "这里有个反直觉的地方"

**引出案例：**
- "举个例子"
- "我遇到过这样一个情况"
- "有个真实的案例可以说明"

### 技巧4：金句制造法

**对比法：**
> "XXX是工具，不是目的"
> "XXX决定下限，YYY决定上限"

**比喻法：**
> "XXX就像YYY，看起来ZZZ，其实AAA"

**递进法：**
> "XXX不只是YYY，更是ZZZ"

**反问法：**
> "XXX真的是YYY吗？不，它是ZZZ"

## 注意事项

### ⚠️ 必须遵守的原则

1. **真实性原则**
   - 不编造数据
   - 不虚构案例
   - 不夸大效果
   - 承认局限性

2. **价值优先原则**
   - 内容必须对读者有用
   - 不为了字数而凑字
   - 不为了爆款而哗众取宠

3. **尊重原创原则**
   - 引用要注明出处
   - 不抄袭他人观点
   - 致敬但不模仿

4. **持续优化原则**
   - 根据反馈迭代
   - 根据数据调整
   - 根据趋势更新

### 💡 建议

1. **先用预设风格，再自定义**
   - 熟悉8种预设风格
   - 找到最适合你的1-2种
   - 在此基础上微调

2. **重视选题分析**
   - 宁可不写，不要乱写
   - 选题对了，成功一半
   - 选题错了，努力白费

3. **保持自己的声音**
